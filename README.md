## To run the app on Xcode Emulator 
1. Clone the source code
> 
      git clone https://github.com/NahSama/React-Native-BookApp.git
2. Install the dependencies
>
      npm install
3. Run the app on Xcode Emulator
>
      npx react-native run-ios
